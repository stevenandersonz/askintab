# Inline LLM Query Extension

A browser extension to ask questions to LLMs (like Grok and ChatGPT) from any webpage.

## Features
- Select text, press `Cmd+K`, ask a question.
- Focus an input, press `Cmd+K`, get answers in the input.
- See all your questions from the popover icon
- Export conversations as plain text.
- Easy to add new LLMs.
- Highlights stay after refresh.

## Supported LLMs
- Grok (xAI)
- ChatGPT (OpenAI)

## Install
1. Clone this repo: git clone https://github.com/stevenandersonz/inline-llm-extension.git

2. Go to `chrome://extensions/` or `about:addons`.
3. Turn on "Developer Mode".
4. Click "Load unpacked" and pick the folder.

## Use
- **Text**: Highlight text, `Cmd+K`, ask, see answer inline.
- **Input**: Focus input, `Cmd+K`, ask, get answer in input.
- **Export**: Click the extension icon -> Select the converation -> Click export.
